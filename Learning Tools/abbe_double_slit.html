The lens is a low-pass filter: it can only keep orders inside the NA.

Abbe's principle, made literal. A mask of slits is not stored as "slits" by the optics — the forward transform to_pupil = FFT rewrites it as a set of spatial-frequency components, the diffraction orders, shown as spikes in panel 1. Each spike is one sine wave (panel 2): the 0 order is the flat DC term (average brightness), the ±1 pair is the fundamental cosine, the ±2 pair a higher harmonic. Abbe's key insight is in panel 3: adding those sine waves back together reconstructs the interference pattern — with only the 0 order the field is flat (no image); add ±1 and the fringes appear; add ±2 and they sharpen toward the true mask. The lens pupil acts as a low-pass filter — orders outside the NA are simply dropped from the sum, so the image is always built from a truncated spectrum. Finally to_real = IFFT turns that captured spectrum into the wafer intensity (panel 4). That FFT → keep-what-fits → IFFT round-trip is the entire imaging model inside the SMO Studio. Physics is exaggerated for intuition, not drawn to scale.
